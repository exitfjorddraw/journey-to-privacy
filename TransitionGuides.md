## Switching from Windows to Linux:
Today this was much easier than it was even 5 years ago due to better hardware support on Linux I think.

- For those that have never used Linux -> try Linux Mint or Ubuntu first (they just work usually)
- Gaming was a blocker for me leaving windows for a long time but most games work now with Steam
  - games like multiplayer esports don't always work but I recommend quitting those anyway for a privacy and life upgrade
  - I was surprised how many games actually performed better (like better graphics and fps even)
- learn about package managers (they are amazing!) like apt-get for finding and installing software quickly
- learn about FOSS (free and open source software) and how to build them yourself for transparency and control

## Switching from Discord to Signal:

- I found this quite easy overall. Signal is much simpler which feels great since discord was getting bloated and annoying.
- Make sure to do a pass through the settings (they aren't long) and set disappearing messages to something like 1-4 weeks.
- I think the devs are worth supporting with a donation (especially if you use it a lot).

## Using paper and pencil for passwords and account names:

- batch generate like 24 strong passwords and 24 random usernames (this should last you at least a year, refresh every few years perhaps)
  - search for password word lists or make your own for making them in a format like this: Username: blah-apple-frog Password: bear-kite-clasp-faint
- list your accounts on a separate sheet and use the line numbers for which user name and password they are
- keep a dice with around 24 sides near where you use them and easily pick new account info as needed
